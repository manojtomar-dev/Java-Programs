# Java Programs

This repository contains a collection of beginner-friendly Java programs.  
These programs helped me practice core Java concepts such as loops, conditions, functions, and number logic.

## üìö Programs Included

1. **Prime Number Checker**
   Checks whether a number is prime or not.

2. **Palindrome Number Checker**  
   Checks if a number reads the same forward and backward.

3. **Armstrong Number Checker**  
   Determines whether a number is an Armstrong number.

4. **LCM Finder**  
   Calculates the Least Common Multiple of two numbers.

5. **HCF Finder**  
   Calculates the Highest Common Factor (GCD) of two numbers.

6. **Strong Number Checker**  
   Verifies whether a number is a Strong Number (sum of factorial of digits).

7. **Nth Prime Number Checker**
   Checks the Nth or desired prime Number.
